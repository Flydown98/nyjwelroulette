2. 새 기능
- 선착순 N명 선정 방식입니다.
- 기본 선정 인원은 30명입니다.
- 선정 완료 후 선정자 명단이 크게 표시됩니다.
- 명단이 길면 마우스 휠 또는 드래그로 확인할 수 있습니다.
- 더블클릭하면 현재 선정자 명단을 복사합니다.
- 누적 선정자가 브라우저 localStorage에 저장됩니다.
- 다음 추첨 시 “이전 선정자 자동 제외”를 켜면 누적 선정자는 자동 제외됩니다.
- “선정자 제외하고 다음 추첨” 버튼을 누르면 바로 다음 회차 준비가 됩니다.
- “선정 기록 초기화” 버튼을 누르면 누적 선정자 기록이 삭제됩니다.

3. CI 이미지
- assets/nyjwel-ci.png 파일을 실제 복지관 CI 이미지로 교체하세요.
- 파일명은 반드시 nyjwel-ci.png로 유지하세요.
- CI는 화면 배경이 아니라 맵 하단 깔때기 근처에 아주 약하게 박히도록 설정되어 있습니다.

4. GitHub Pages 주소
- 저장소 이름이 Flydown98/nyjwelroulette 기준으로 맞춰져 있습니다.
- build public-url은 /nyjwelroulette/ 입니다.
- 저장소 이름을 바꾸면 package.json의 build 주소도 함께 바꿔야 합니다.

5. 행사 사용 팁
- 참가자 이름은 코드에 저장하지 말고, 행사 당일 입력창에 붙여넣으세요.
- 브라우저를 바꿔 열면 누적 선정 기록이 이어지지 않습니다.
- 기록을 완전히 초기화하려면 화면의 “선정 기록 초기화” 버튼을 누르세요.
- 페이지가 바뀌지 않으면 Ctrl + F5로 강력 새로고침하세요.
